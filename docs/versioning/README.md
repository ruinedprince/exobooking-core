# Versionamento

Documentação e convenções de versionamento do projeto ExoBooking Core.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| **[VERSIONING.md](VERSIONING.md)** | Guia de versionamento semântico (x.y.z) e regras MAJOR/MINOR/PATCH |
| **`../../VERSION`** | Versão atual do projeto (fonte única na raiz) |
| **`../../CHANGELOG.md`** | Histórico de alterações por versão e data |

## Versão atual

A versão é definida no arquivo **`VERSION`** na raiz do projeto e deve ser replicada no cabeçalho do plugin (`exobooking-core/exobooking-core.php`).
